# validZT

Инструмент для проверки и очистки текста перед отправкой письма через [zt.ru](https://zt.ru).

## Что это

zt.ru отклоняет или искажает некоторые Unicode-символы, которые выглядят нормально в любом редакторе - минус вместо дефиса, знак рубля, пробелы нестандартной ширины, эмодзи. validZT находит такие символы, подсвечивает в тексте и заменяет на допустимые аналоги одной кнопкой.

Дополнительно приложение оценивает количество страниц, на которые zt.ru разобьёт письмо.

## Конфиденциальность

**Текст не покидает устройство.** Приложение работает полностью в браузере. После первого открытия все файлы сохраняются локально и приложение работает без интернета.
 

## Два формата доставки

В нижней строке экрана есть кнопка переключения формата:

- **ZT** — стандартный формат ZT (пропорциональный шрифт, рамка заголовка и подвала). Используется по умолчанию. Примерно 2580 символов кириллицей на страницу.
- **ПР** — формат для учреждений с типом доставки Почта России (моноширинный шрифт, упрощённый заголовок). Примерно 2470 символов на страницу.

Выбранный формат сохраняется между сессиями.

## Как пользоваться

1. Вставьте текст письма кнопкой **Вставить** или через Ctrl+V.
2. Приложение сразу покажет проблемные символы, подсветив их красным.
3. Нажмите **Исправить** — символы будут заменены, изменения подсветятся зелёным.
4. Убедитесь, что замены корректны. При необходимости нажмите «↩» (отмена).
5. Нажмите **Копировать** и вставьте готовый текст в zt.ru.

## Установка офлайн (PWA)

Нажмите кнопку **✈ offline** в правом верхнем углу — там подробная инструкция для Android, iPhone и компьютера.

После установки приложение запускается как обычное: без браузера, без адресной строки, с иконкой на рабочем столе.
